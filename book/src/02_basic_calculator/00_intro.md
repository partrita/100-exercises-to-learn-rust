# 기본 계산기 만들기

이 장에서는 Rust를 **계산기**처럼 활용하는 방법을 배웁니다. 단순해 보일 수 있지만, 이 과정을 통해 다음과 같은 Rust의 핵심 기초를 탄탄히 다질 수 있습니다.

- 함수를 정의하고 호출하는 방법
- 변수를 선언하고 사용하는 방법
- 기본 데이터 타입 (정수 및 불리언)
- 산술 연산자 (오버플로우 및 언더플로우 동작 포함)
- 비교 연산자
- 제어 흐름(Control Flow)
- 패닉(Panics)

몇 가지 연습 문제를 직접 풀어보며 Rust의 기초를 몸소 익히게 될 것입니다. 나중에 특성(Traits)이나 소유권(Ownership) 같은 더 복잡한 주제로 넘어갔을 때, 기본적인 구문이나 세부 사항에 얽매이지 않고 새로운 개념에만 집중할 수 있는 든든한 밑거름이 되어줄 것입니다.
